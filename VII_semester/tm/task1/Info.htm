Kalbos semantika laisva forma

Open eBook Publication Structure (OEBPS) specifikacija skirta apraðyti informacijai apie elektronines knygas.

OEBS dokumentas yra atvaizduojamas XML formatu. Vienas dokumentas turi turëti tik vienà package þymæ, kurioje apraðomi dokumentai sudarantys publikacijà.

Package þymæ sudaro:

metadata þymë
apraðomi publikacijos meta duomenys
manifest þymë
apraðomas sàraðas dokumentø, sudaranèiø publikacijà
spine þymë
apraðoma publikacijà sudaranèiø dokumentø skaitymo tvarka
tours þymë
apraðomos publikacijà sudaranèiø dokumentø alternatyvios skaitymo tvarkos
guide þymë
apraðomos nuorodos á elementus, tokius kaip turinys, bibliografija ir pan.

Atlikto darbo palyginimas

File oeb.bnf pateiktas sintaksës apraðymas yra daug konkretesnis nei pateiktame pavyzdyje. Pavyzdyje sintaksei nëra priskiriamos þymës, kurios yra naudojamos dokumente. Taip pat, pavyzdyje trûkstà kai kuriø kairës reikðmës apibreþimø. Pvz trûktsa Parametrø þodþiai apibrëþimo.
